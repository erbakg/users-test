# Users Catalog

A user catalog page built with React + TypeScript as a test assignment.

Features:
- User list with avatar, name, email, phone, age, and company info
- Search by name (debounced)
- Pagination

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v4

## API

[DummyJSON Users API](https://dummyjson.com/docs/users)

## Getting Started

```bash
# Install dependencies
yarn install

# Run dev server
yarn dev

# Build for production
yarn build
```

The app will be available at `http://localhost:5173`.
